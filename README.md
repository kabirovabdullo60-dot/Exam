# Exam
# Кабинет метролога

## Дар бораи лоиҳа

Ин лоиҳа веб-сайти "Кабинет метролога" мебошад, ки барои осон кардани кори лабораторияҳои метрологӣ сохта шудааст. Дар сомона маълумот дар бораи хизматрасониҳо, имкониятҳои система ва афзалиятҳои истифодаи он пешниҳод карда мешаванд.

Лоиҳа ҳамчун веб-интерфейси мутобиқшаванда (Responsive Design) таҳия шудааст ва дар дастгоҳҳои гуногун дуруст намоиш дода мешавад.

---

## Имкониятҳои асосӣ

- Намоиши маълумот дар бораи хизматрасониҳои система
- Интерфейси оддӣ ва фаҳмо барои истифодабарандагон
- Блокҳои иттилоотӣ бо тасвирҳо ва нишонаҳо
- Қисмати тарифҳо ва имкониятҳои система
- Қисмати кӯмак ва дастгирии техникӣ
- Дизайни мутобиқшаванда барои экранҳои гуногун

---

## Дастгоҳҳои дастгиришаванда

Лоиҳа барои се намуди дастгоҳ мутобиқ карда шудааст:

- 💻 Laptop/Desktop
- 📱 Mobile Phone
- 📟 Tablet

Ҳамаи элементҳо вобаста ба андозаи экран ҷойгиршавии худро тағйир медиҳанд, то истифодаи сайт қулай бошад.

---

## Технологияҳои истифодашуда

- HTML5
- CSS3
- Flexbox
- Media Queries

---

## Ҳадафи лоиҳа

Мақсади асосии ин лоиҳа омӯзиши сохтани веб-сайтҳои Responsive ва кор бо HTML ва CSS мебошад. Ҳангоми иҷрои лоиҳа таҷрибаи кор бо тарҳбандии саҳифа, мутобиқсозии интерфейс барои дастгоҳҳои гуногун ва тарҳрезии веб гирифта шуд.

---

## Намоиши лоиҳа

### Desktop Version
Саҳифа барои экранҳои калон бо ҷойгиршавии пурраи ҳамаи қисмҳо таҳия шудааст.

### Tablet Version
Тарҳ ба андозаи планшет мутобиқ карда шуда, баъзе элементҳо ҷойгиршавии худро тағйир медиҳанд.

### Mobile Version
Барои телефонҳо ҳамаи блокҳо ба шакли амудӣ ҷойгир шудаанд, то хондан ва истифода осон бошад.

---

## Муаллиф

Лоиҳа ҳамчун кори таълимӣ барои омӯзиши Frontend Development таҳия шудааст.
